# Credit pipeline

___

## Setup step by step:

1. ```git clone https://github.com/anko011/credit_pipeline```
2. ```cd ./credit_pipeline```
3. ```pip -m venv venv```
4. ```./venv/Scripts/activate```
5. ```pip install -r ./requirements.txt```

## Launch

1. ```cd ..```
2. ```python -m credit_pipeline```
